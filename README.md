JS-Fundamentals

A concise collection of JavaScript exercises and examples designed to help you practice and strengthen core JavaScript concepts.

   Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Ntinyari1/JS-Fundamentals.git
   cd JS-Fundamentals
   ```

2. Run any of the JavaScript files:
   ```bash
   node path/to/file.js
   ```

  Repository Structure

-exercises/ Practice problems grouped by topic.
-solutions/ — Reference solutions (if provided).
-examples/ — Demonstrations of fundamental concepts.
